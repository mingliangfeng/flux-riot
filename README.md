# flux-riot

> use riot with flux architecture


## Running the code

```bash
$ git clone https://github.com/mingliangfeng/flux-riot.git
$ cd flux-riot
$ npm install
$ npm test
$ npm start

```


## Run TODO example

```bash
$ cd flux-riot/examples/todo
$ npm install
$ npm test
$ npm start

```
